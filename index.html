柵格排列

響應式柵格排列，呈現方式類似表格。
/* Small devices (mobile phone, 768px bellow) */
@ media (max-width: @screen-xs-max) { ... }

/* Small devices (tablets, 768px and up) */
@ media (min-width: @screen-sm-min) and (max-width: @screen-sm-max) { ... }

/* Medium devices (desktops, 992px and up) */
@ media (min-width: @screen-md-min) and (max-width: @screen-md-max) { ... }

/* Large devices (large desktops, 1200px and up) */
@ media (min-width: @screen-lg-min) { .. . }

範例

col-xs-*
(*代表合併欄位數，有點類似colspan)
超小屏幕設備 手機(<768px)
總是水平排列
最大.container宽度 None (自动)
最大列宽 auto
col-sm-*
(*代表合併欄位數，有點類似colspan)
小屏幕设备 平板 (≥768px)
開始是堆疊在一起的，超過這些閾值將變為水平排列
最大.container宽度 750px
最大列宽 60px
col-md-*
(*代表合併欄位數，有點類似colspan)
中等屏幕設備 桌面(≥992px)
開始是堆疊在一起的，超過這些閾值將變為水平排列
最大.container宽度 970px
最大列宽 78px
col-lg-*
(*代表合併欄位數，有點類似colspan)
大屏幕設備 桌面(≥1200px)
開始是堆疊在一起的，超過這些閾值將變為水平排列
最大.container宽度 1170px
最大列宽 95px

表單

表單元件屬性

支持HTML5所有的屬性類別，例如:text、password、datetime、datetime-local、date、month、time、week、number、email、url、search、tel和color。

Label

sr-only: 可以隱藏元件。
建議ㄧ定要加label標籤，如果不想顯示，可以用sr-only隱藏label。
label 的 for屬性功能，可以跟據id綁定表單元素。 

          <label for='target'>Label</label>
          <input type="text" id="target" class="form-control"/>

表單基本設定

form-group:在此標籤中設定表單樣式，效果較佳。
form-control:將表單標籤套用樣式。預設寬度為100%。
btn btn-default:按鈕樣式。

表單排列設定

併排排列

在form標籤設置form-inline:可使其內容左對齊並且表現為inline-block級別的控件。

水平排列

在form標籤設置form-horizontal:可使其內容水平排列。
form-group具有row功能，因此可以直接使用col-sm或col-sm-offset控制label
control-label使label靠右
col-sm或col-sm-offset控制input

按鈕

ㄧ般設定

可以為<a>、<button>或<input>標籤添加按鈕樣式。
為了符合跨瀏覽器，建議使用<bottom>元素，確保一致性。
快速設定按鈕樣式:
btn-default預設
btn-primary藍色
btn-success綠色
btn-info藍色
btn-warning橘色
btn-danger紅色
btn-link一般超連結 


            <button type="button" class="btn btn-default">Default</button>
            <button type="button" class="btn btn-primary">Primary</button>
            <button type="button" class="btn btn-success">Success</button>
            <button type="button" class="btn btn-info">Info</button>
            <button type="button" class="btn btn-warning">Warning</button>
            <button type="button" class="btn btn-danger">Danger</button>
            <button type="button" class="btn btn-link">Link</button>
